Detailed Description

Support functions for the glthread feature of Mesa.

In multicore systems, many applications end up CPU-bound with about half their time spent inside their rendering thread and half inside Mesa. To alleviate this, we put a shim layer in Mesa at the GL dispatch level that quickly logs the GL commands to a buffer to be processed by a worker thread.
