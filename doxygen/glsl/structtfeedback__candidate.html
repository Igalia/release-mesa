Detailed Description

Data structure describing a varying which is available for use in transform feedback.

For example, if the vertex shader contains: 

struct S {
  vec4 foo;
  float[3] bar;
};

varying S[2] v;

Then there would be tfeedback_candidate objects corresponding to the following varyings: 

v[0].foo
v[0].bar
v[1].foo
v[1].bar
