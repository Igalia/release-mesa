GLboolean _mesa_readpixels_needs_slow_path ( const struct gl_context *  ctx,
GLenum  format,
GLenum  type,
GLboolean  uses_blit 
)

Return true if memcpy cannot be used for ReadPixels.

If uses_blit is true, the function returns true if a simple 3D engine blit cannot be used for ReadPixels packing.

NOTE: This doesn't take swizzling and format conversions between the readbuffer and the pixel pack buffer into account.
