§ brw_render_state_finished()

void brw_render_state_finished ( struct brw_context *  brw)

Clear dirty bits to account for the fact that the state emitted by brw_upload_render_state() has been committed to the hardware.

This is a separate call from brw_upload_render_state() because it's possible that after the call to brw_upload_render_state(), we will discover that we've run out of aperture space, and need to rewind the batch buffer to the state it had before the brw_upload_render_state() call.
