Apply stencil test to an array of stencil values (before depth buffering).

For the values that fail, we'll apply the GL_STENCIL_FAIL operator to the stencil values.

Parameters
face0 or 1 for front or back-face polygons
nnumber of pixels in the array
stencilarray of [n] stencil values (in/out)
maskarray [n] of flag: 0=skip the pixel, 1=stencil the pixel, values are set to zero where the stencil test fails.
stridestride between stencil values
Returns
GL_FALSE = all pixels failed, GL_TRUE = zero or more pixels passed.
