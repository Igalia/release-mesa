Detailed Description

Propagate the "invariant" and "precise" qualifiers to variables used to compute invariant or precise values.

The GLSL spec (depending on what version you read) says, among the conditions for getting bit-for-bit the same values on an invariant output:

"All operations in the consuming expressions and any intermediate expressions must be the same, with the same order of operands and same associativity, to give the same order of evaluation."

This effectively means that if a variable is used to compute an invariant value then that variable becomes invariant. The same should apply to the "precise" qualifier.
