§ nir_gs_count_vertices()

int nir_gs_count_vertices ( const nir_shader *  shader)

If a geometry shader emits a constant number of vertices, return the number of vertices.

Otherwise, return -1 (unknown).

This only works if you've used nir_lower_gs_intrinsics() to do vertex counting at the NIR level.
