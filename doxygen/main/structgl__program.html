§ ImageAccess

GLenum gl_program::ImageAccess[MAX_IMAGE_UNIFORMS]

Access qualifier specified in the shader for each image uniform index.

Either GL_READ_ONLY, GL_WRITE_ONLY or GL_READ_WRITE.

It may be different, though only more strict than the value of gl_image_unit::Access for the corresponding image unit.

§ ImageUnits

GLubyte gl_program::ImageUnits[MAX_IMAGE_UNIFORMS]

Map from image uniform index to image unit (set by glUniform1i())

An image uniform index is associated with each image uniform by the linker. The image index associated with each uniform is stored in the gl_uniform_storage::image field.
