unsigned  components,
void *  mem_ctx 
)

Generate a comparison value for a block of indices.

Lowering passes for non-constant indexing of arrays, matrices, or vectors can use this to generate blocks of index comparison values.

Parameters
instructionsList where new instructions will be appended
indexir_variable containing the desired index
baseBase value for this block of comparisons
componentsNumber of unique index values to compare. This must be on the range [1, 4].
mem_ctxralloc memory context to be used for all allocations.
Returns
An ir_rvalue that must be cloned for each use in conditional assignments, etc.

§ do_common_optimization()

bool do_common_optimization ( exec_list *  ir,
bool  linked,
bool  uniform_locations_assigned,
const struct gl_shader_compiler_options *  options,
bool  native_integers 
)

Do the set of common optimizations passes.

Parameters
irList of instructions to be optimized
linkedIs the shader linked? This enables optimizations passes that remove code at global scope and could cause linking to fail.
uniform_locations_assignedHave locations already been assigned for uniforms? This prevents the declarations of unused uniforms from being removed. The setting of this flag only matters if linked is true.
optionsThe driver's preferred shader options.
native_integersSelects optimizations that depend on the implementations supporting integers natively (as opposed to supporting integers in floating point registers).
