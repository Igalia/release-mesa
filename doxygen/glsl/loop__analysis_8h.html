Find an initializer of a variable outside a loop.

Works backwards from the loop to find the pre-loop value of the variable. This is used, for example, to find the initial value of loop induction variables.

Parameters
loopLoop where var is an induction variable
varVariable whose initializer is to be found
Returns
The ir_rvalue assigned to the variable outside the loop. May return NULL if no initializer can be found.

§ set_loop_controls()

bool set_loop_controls ( exec_list *  instructions,
loop_state *  ls 
)

Fill in loop control fields.

Based on analysis of loop variables, this function tries to remove redundant sequences in the loop of the form

(if (expression bool ...) (break))

For example, if it is provable that one loop exit condition will always be satisfied before another, the unnecessary exit condition will be removed.
