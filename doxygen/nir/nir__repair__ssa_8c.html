§ nir_repair_ssa()

bool nir_repair_ssa ( nir_shader *  shader)

This pass can be used to repair SSA form in a shader.

Sometimes a transformation (such as return lowering) will have to make changes to a shader which, while still correct, break some of NIR's SSA invariants. This pass will insert ssa_undefs and phi nodes as needed to get the shader back into SSA that the validator will like.
