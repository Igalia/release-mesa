§ pma_fix_enable()

static bool pma_fix_enable ( const struct brw_context *  brw)
static

Should we set the PMA FIX ENABLE bit?

To avoid unnecessary depth related stalls, we need to set this bit. However, there is a very complicated formula which governs when it is legal to do so. This function computes that.

See the documenation for the CACHE_MODE_1 register, bit 11.
