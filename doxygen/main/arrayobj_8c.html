Detailed Description

Implementation of Vertex Array Objects (VAOs), from OpenGL 3.1+, the GL_ARB_vertex_array_object extension, or the older GL_APPLE_vertex_array_object extension.

Todo:
The code in this file borrows a lot from bufferobj.c. There's a certain amount of cruft left over from that origin that may be unnecessary.
