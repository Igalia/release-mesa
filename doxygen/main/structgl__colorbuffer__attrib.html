§ _AdvancedBlendMode

enum gl_advanced_blend_mode gl_colorbuffer_attrib::_AdvancedBlendMode

Which advanced blending mode is in use (or BLEND_NONE).

KHR_blend_equation_advanced only allows advanced blending with a single draw buffer, and NVX_blend_equation_advanced_multi_draw_buffer still requires all draw buffers to match, so we only need a single value.
