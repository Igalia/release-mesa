Creates a region containing the push constants for the CS on gen7+.

Push constants are constant values (such as GLSL uniforms) that are pre-loaded into a shader stage's register space at thread spawn time.

For other stages, see brw_curbe.c:brw_upload_constant_buffer for the equivalent gen4/5 code and gen6_vs_state.c:gen6_upload_push_constants for gen6+.
