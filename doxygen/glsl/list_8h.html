Detailed Description

Doubly-linked list abstract container type.

Each doubly-linked list has a sentinel head and tail node. These nodes contain no data. The head sentinel can be identified by its prev pointer being NULL. The tail sentinel can be identified by its next pointer being NULL.

A list is empty if either the head sentinel's next pointer points to the tail sentinel or the tail sentinel's prev poiner points to the head sentinel. The head sentinel and tail sentinel nodes are allocated within the list structure.

Do note that this means that the list nodes will contain pointers into the list structure itself and as a result you may not realloc() an exec_list or any structure in which an exec_list is embedded.
