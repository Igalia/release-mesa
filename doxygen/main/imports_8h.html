Detailed Description

Standard C library function wrappers.

This file provides wrappers for all the standard C library functions like malloc(), free(), printf(), getenv(), etc.

§ _mesa_next_pow_two_32()

static int32_t _mesa_next_pow_two_32 ( uint32_t  x)
inlinestatic

Round given integer to next higer power of two If X is zero result is undefined.

Source for the fallback implementation is Sean Eron Anderson's webpage "Bit Twiddling Hacks" http://graphics.stanford.edu/~seander/bithacks.html

When using builtin function have to do some work for case when passed values 1 to prevent hiting undefined result from __builtin_clz. Undefined results would be different depending on optimization level used for build.
