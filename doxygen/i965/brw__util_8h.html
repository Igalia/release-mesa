GLenum brw_fix_xRGB_alpha ( GLenum  function)

Modify blend function to force destination alpha to 1.0.

If function specifies a blend function that uses destination alpha, replace it with a function that hard-wires destination alpha to 1.0. This is used when rendering to xRGB targets.
