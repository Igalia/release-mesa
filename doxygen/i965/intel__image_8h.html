Detailed Description

Structure definitions and prototypes for __DRIimage, the driver-private structure backing EGLImage or a drawable in DRI3.

The __DRIimage is passed around the loader code (src/glx and src/egl), but it's opaque to that code and may only be accessed by loader extensions (mostly located in intel_screen.c).
