§ _mesa_get_format_swizzle()

void _mesa_get_format_swizzle ( mesa_format  format,
uint8_t  swizzle_out[4] 
)

Returns the an array of four numbers representing the transformation from the RGBA or SZ colorspace to the given format.

For array formats, the i'th RGBA component is given by:

if (swizzle[i] <= MESA_FORMAT_SWIZZLE_W) comp = data[swizzle[i]]; else if (swizzle[i] == MESA_FORMAT_SWIZZLE_ZERO) comp = 0; else if (swizzle[i] == MESA_FORMAT_SWIZZLE_ONE) comp = 1; else if (swizzle[i] == MESA_FORMAT_SWIZZLE_NONE) // data does not contain a channel of this format

For packed formats, the swizzle gives the number of components left of the least significant bit.

Compressed formats have no swizzle.
