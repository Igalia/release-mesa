Detailed Description

Replaces references to the "orig" variable with a clone of "repl.".

From the spec, opaque types can appear in the tree as function (non-out) parameters and as the result of array indexing and structure field selection. In our builtin implementation, they also appear in the sampler field of an ir_tex instruction.
