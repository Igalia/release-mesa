Do a dead code pass over instructions and everything that instructions references.

Note that this will remove assignments to globals, so it is not suitable for usage on an unlinked instruction stream.

Does a dead code pass on the functions present in the instruction stream.

This is suitable for use while the program is not linked, as it will ignore variable declarations (and the assignments to them) for variables with global scope.
