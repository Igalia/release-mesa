§ soft_renderbuffer_storage()

static GLboolean soft_renderbuffer_storage ( struct gl_context *  ctx,
struct gl_renderbuffer *  rb,
GLenum  internalFormat,
GLuint  width,
GLuint  height 
)
static

Functions for allocating/managing software-based renderbuffers.

Also, routines for reading/writing software-based renderbuffer data as ubytes, ushorts, uints, etc. This is a software fallback for the gl_renderbuffer->AllocStorage function. Device drivers will typically override this function for the buffers which it manages (typically color buffers, Z and stencil). Other buffers (like software accumulation and aux buffers) which the driver doesn't manage can be handled with this function.

This one multi-purpose function can allocate stencil, depth, accum, color or color-index buffers!
