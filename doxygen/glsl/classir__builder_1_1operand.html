Detailed Description

This little class exists to let the helper expression generators take either an ir_rvalue * or an ir_variable * to be automatically dereferenced, while still providing compile-time type checking.

You don't have to explicitly call the constructor – C++ will see that you passed an ir_variable, and silently call the operand(ir_variable *var) constructor behind your back.
