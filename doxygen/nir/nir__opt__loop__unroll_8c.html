static void complex_unroll ( nir_loop *  loop,
nir_loop_terminator *  unlimit_term,
bool  limiting_term_second 
)
static

Unroll a loop with two exists when the trip count of one of the exits is unknown.

If continue_from_then is true, the loop is repeated only when the "then" branch of the if is taken; otherwise it is repeated only when the "else" branch of the if is taken.

For example, if the input is: 

 loop {
    ...phis/condition...
    if condition {
       ...then instructions...
    } else {
       ...continue instructions...
       break
    }
    ...body...
 }

And the iteration count is 3, and unlimit_term->continue_from_then is true, then the output will be: 

 ...condition...
 if condition {
    ...then instructions...
    ...body...
    if condition {
       ...then instructions...
       ...body...
       if condition {
          ...then instructions...
          ...body...
       } else {
          ...continue instructions...
       }
    } else {
       ...continue instructions...
    }
 } else {
    ...continue instructions...
 }

static void simple_unroll ( nir_loop *  loop)
static

Unroll a loop where we know exactly how many iterations there are and there is only a single exit point.

Note here we can unroll loops with multiple theoretical exits that only have a single terminating exit that we always know is the "real" exit. 

loop {
    ...instrs...
}

And the iteration count is 3, the output will be: 

...instrs... ...instrs... ...instrs...
