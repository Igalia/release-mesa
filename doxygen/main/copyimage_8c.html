Prepare the source or destination resource.

This involves error checking and returning the relevant gl_texture_image or gl_renderbuffer. Note that one of the resulting tex_image or renderbuffer pointers will be NULL and the other will be non-null.

Parameters
namethe texture or renderbuffer name
targetOne of GL_TEXTURE_x target or GL_RENDERBUFFER
levelmipmap level
zsrc or dest Z
depthnumber of slices/faces/layers to copy
tex_imagereturns a pointer to a texture image
renderbufferreturns a pointer to a renderbuffer
Returns
true if success, false if error
