Detailed Description

Describes a set of glBitmap display lists which live in a texture atlas.

The idea is when we see a code sequence of glListBase(b), glCallLists(n) we're probably drawing bitmap font glyphs. We try to put all the bitmap glyphs into one texture map then render the glCallLists as a textured quadstrip.
