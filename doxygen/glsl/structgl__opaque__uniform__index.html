§ index

uint8_t gl_opaque_uniform_index::index

Base opaque uniform index.

If gl_uniform_storage::base_type is an opaque type, this represents its uniform index. If gl_uniform_storage::array_elements is not zero, the array will use opaque uniform indices index through index + gl_uniform_storage::array_elements - 1, inclusive.

Note that the index may be different in each shader stage.
