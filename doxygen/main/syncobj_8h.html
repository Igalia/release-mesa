§ _mesa_get_and_ref_sync()

struct gl_sync_object* _mesa_get_and_ref_sync ( struct gl_context *  ctx,
GLsync  sync,
bool  incRefCount 
)

Check if the given sync object is:

  • non-null
  • not in sync objects hash table
  • type is GL_SYNC_FENCE
  • not marked as deleted

Returns the internal gl_sync_object pointer if the sync object is valid or NULL if it isn't.

If "incRefCount" is true, the reference count is incremented, which is normally what you want; otherwise, a glDeleteSync from another thread could delete the sync object while you are still working on it.
