Detailed Description

Per-vertex functions.

These are the functions which can appear between glBegin and glEnd. Depending on whether we're inside or outside a glBegin/End pair and whether we're in immediate mode or building a display list, these functions behave differently. This structure allows us to switch between those modes more easily.

Generally, these pointers point to functions in the VBO module.
