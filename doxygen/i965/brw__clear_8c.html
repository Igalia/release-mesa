§ brw_fast_clear_depth()

static bool brw_fast_clear_depth ( struct gl_context *  ctx)
static

Implements fast depth clears on gen6+.

Fast clears basically work by setting a flag in each of the subspans represented in the HiZ buffer that says "When you need the depth values for this subspan, it's the hardware's current clear value." Then later rendering can just use the static clear value instead of referencing memory.

The tricky part of the implementation is that you have to have the clear value that was used on the depth buffer in place for all further rendering, at least until a resolve to the real depth buffer happens.
