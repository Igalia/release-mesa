Detailed Description

This file implements a simple program cache for 965.

The consumers can query the hash table of programs using a cache_id and program key, and receive the corresponding program buffer object (plus associated auxiliary data) in return. Objects in the cache may not have relocations (pointers to other BOs) in them.

The inner workings are a simple hash table based on a CRC of the key data.

Replacement is not implemented. Instead, when the cache gets too big we throw out all of the cache data and let it get regenerated.
