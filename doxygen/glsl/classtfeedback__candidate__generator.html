Detailed Description

Visitor class that generates tfeedback_candidate structs describing all possible targets of transform feedback.

tfeedback_candidate structs are stored in the hash table tfeedback_candidates, which is passed to the constructor. This hash table maps varying names to instances of the tfeedback_candidate struct.
