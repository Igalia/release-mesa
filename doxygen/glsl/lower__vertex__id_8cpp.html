Detailed Description

There exists hardware, such as i965, that does not implement the OpenGL semantic for gl_VertexID.

Instead, that hardware does not include the value of basevertex in the gl_VertexID value. To implement the OpenGL semantic, we'll have to convert gl_Vertex_ID to gl_VertexIDMESA+gl_BaseVertexMESA.
