Detailed Description

Renderbuffers represent drawing surfaces such as color, depth and/or stencil.

A framebuffer object has a set of renderbuffers. Drivers will typically derive subclasses of this type.

§ NeedsFinishRenderTexture

GLboolean gl_renderbuffer::NeedsFinishRenderTexture

True for renderbuffers that wrap textures, giving the driver a chance to flush render caches through the FinishRenderTexture hook.

Drivers may also set this on renderbuffers other than those generated by glFramebufferTexture(), though it means FinishRenderTexture() would be called without a rb->TexImage.
