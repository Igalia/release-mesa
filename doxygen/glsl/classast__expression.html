§ non_lvalue_description

const char* ast_expression::non_lvalue_description

For things that can't be l-values, this describes what it is.

This text is used by the code that generates IR for assignments to detect and emit useful messages for assignments to some things that can't be l-values. For example, pre- or post-incerement expressions.

Note
This pointer may be NULL.
